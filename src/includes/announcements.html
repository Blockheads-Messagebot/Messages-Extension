Announcements

  • These are sent according to a regular schedule.
  • If you have one announcement, it is sent every X minutes, if you have two, then the first is sent at X minutes, and the second is sent X minutes after the first. Change X in the settings tab. Once the bot reaches the end of the list, it starts over at the top.
  • If no players are online when a message is scheduled to be sent, no message will be sent and the next message will reset to the first message.
